## 생성자와 함수 차이점.

```
1. 생성자 : 초기화에 목적을 둠.
2. 함수 : 로직을 처리 or 리턴에 목적을 둠.
3. 생성자는 리턴타입이 없음.
4. 생성자 이름이 클래스 이름과 동일해야함.
```

## 자바특징(static 관련)

```java
static main(){
    //static 함수에는 static함수만 올 수 있다.
    //new로 호출한 클래스의 함수도 사용 가능
    //(동적타입도 가능!)
}

ex)
public class 클래스이름{
void test(){

}
static main(){
    test() // 사용 못함->test()의 함수가 static이 아니라서.

    //test()함수를 불러올 수 있는 방법
    // 방법1.
    // 함수에 static을 붙여주면 된다.

    // 방법2.
    클래스이름 class = new 클래스이름();
    class.test() //클래스를 호출한뒤 함수를 불러오면 사용가능!
}
}
```
