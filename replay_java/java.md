```
- 클래스(객체)에 대한 개념이 부족하면 상속이 어려울 수 있음.
```

# 데이터 타입

```
자바에서는 동적 데이터타입과 정적 데이터타입을 저장하는 공간이 다르다
.
1. 정적 타입(int, double, String, char, 배열...)

2. 동적 타입(String, class,  배열...)
    2-1) 동적 데이터타입 저장공간 = *Heap Area(메모리)
    2-2) 동적타입을 관리하는 영역을 *Heap Area이라고 한다.
    2-3) 동적타입은 Heap Area에 저장을 한다.
```

# 자바

```
- 계층 구조
    프로젝트 -> 패키지(default) -> 클래스파일

    1. 프로젝트 생성
        1-1) 프로젝트이름은 대문자 or 소문자
        1-2) 복합명사일 경우 ' _ ' 사용 (정답은 아님)

    2. 패키지 생성(생략가능)
        2-1) 패키지 이름은 소문자
        2-2) 복합명사일경우 카멜표기법 사용
    (프로젝트에 바로 클래스를 만들면 dafault패키지가 생성된다.)

    3. 클래스 파일 생성
        3-1) 클래스 이름은 항상 맨앞은 대문자 사용

- 현재 사용 자바 버전 = 12버전
   ( 하위버전을 사용하면 상위버전을 사용하지 못하지만 상위버전 사용시 하위버전 사용 가능)
```

## 생성자와 함수의 차이

```
1. 생성자는 return값이 없음
2. 함수는 return 값이 있어야 사용 가능.
3. 사용 목적
    3-1) 생성자 : 생성자는 값을 초기화하는게 목적
    3-2) 함수 : 함수는 파라미터에 값을 받아서 결과를 처리하는게 목적
```

## new 호출

```
- new를 이용해서 클래스를 호출할 때 생성자가 자동으로 호출됨.
- new로 불러온 변수 재활용 가능!
```
